# agent-society-sim

基于大语言模型的多智能体社会模拟系统。每个智能体拥有对外的公开人设与仅自身可见的私密目标和动机,在共享场景中自主交互;开发者通过命令行界面以观察者视角,实时查看由此涌现的群体行为与关系演变。

引擎与场景解耦:引擎通用,世界观与角色作为可替换的数据载入。预览中的「樱丘高中·文化祭」仅为当前示例场景。

> 状态:早期开发中。首要目标为 multi-agent 架构的工程实践。
